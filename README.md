# Article Writer

リファレンス記事のスタイルを分析し、同じトーンで新しい記事を生成するツール。
AI には Claude（Anthropic）を使用。

## セットアップ

```bash
# 依存関係のインストール
pip3 install -r requirements.txt

# 環境変数の設定
cp .env.example .env
# .env ファイルを開いて ANTHROPIC_API_KEY を設定する

# 起動
python3 app.py
```

ブラウザで http://localhost:5050 を開く。

## 使い方

1. **コンテキスト登録** ── 再現したいスタイルの記事を最大3つ貼り付ける
2. **スタイル分析** ── Claude が記事の構成・文体・リズムを分析
3. **執筆指示** ── タイトルと内容メモを入力
4. **インタビュー** ── Claude が素材を深掘りする質問をする
5. **記事生成** ── スタイルを再現した記事がHTML形式で出力される
6. **コピー** ── 「記事をコピー」ボタンでX記事エディタに貼り付け可能
